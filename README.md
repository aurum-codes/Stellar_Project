# Stellar_Project - ScholrChain

A decentralized student credential and enrollment system built on the **Stellar Network** using **Soroban Smart Contracts**. 

This project demonstrates how to securely store and retrieve student records on a blockchain using a modern web interface. It serves as an excellent foundational project for understanding decentralized applications (dApps), combining a robust Rust backend with a fast Next.js frontend.
