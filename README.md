**ScholrChain** is a decentralized student credential and enrollment system built on the **Stellar Network** utilizing **Soroban Smart Contracts**. 

This project demonstrates how to securely store, manage, and retrieve student records on a blockchain infrastructure using a modern web interface. It combines a robust Rust-based smart contract backend with a fast, responsive Next.js frontend.

## ✨ Features

* **Decentralized Storage:** Securely record student enrollment and credentials on the Stellar blockchain.
* **Smart Contract Integration:** Utilizes Soroban (Rust) to handle backend logic, ensuring data immutability and transparency.
* **Modern Web Interface:** A seamless user experience built with Next.js, TypeScript, and CSS.
* **Wallet Integration:** Allows users to interact with the blockchain directly from their browser.

## 🛠️ Tech Stack

### Smart Contract (Backend)
* **Language:** Rust
* **Network:** Stellar (Soroban)

### Client (Frontend)
* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** CSS
